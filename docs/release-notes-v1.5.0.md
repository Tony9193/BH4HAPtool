# BH4HAPtool v1.5.0 更新内容

发布日期：2026-04-18

## 版本亮点

- 记录中心升级
  - 新增使用统计榜，支持查看高频工具与累计使用占比
  - 新增最近使用序列，快速回顾近期工具链
  - 概览卡新增累计启动次数、上次使用时间、不同工具使用数
- 成就系统升级
  - 新增成就总览（已解锁数量/总数量）
  - 每个成就支持进度条与进度值显示（例如 3/5、650/1000）
  - 成就状态与工具统计、小游戏记录联动更新
- 首页使用统计增强
  - 新增“热度榜”分区，展示高频工具和累计使用次数
  - 与收藏、最近使用形成互补入口，提升回访效率

## 体验改进

- 番茄钟记录在记录中心改为按自然日计算“今日完成番茄”，跨天显示更准确
- 记录页面信息结构优化，支持更快完成活动复盘
- 离线版本更新说明新增 `v1.4.0` 与 `v1.5.0`

## 建议测试项

- 首页热度榜在多次启动工具后是否实时更新
- 记录中心使用统计与最近使用数据是否与首页一致
- 使用 5 个不同模块后“全能工具人”成就是否解锁
- 俄罗斯方块达到 1000 分后成就进度与解锁状态是否正确
- 番茄钟跨天后“今日完成番茄”是否正确显示为 0

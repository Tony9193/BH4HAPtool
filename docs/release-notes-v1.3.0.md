# BH4HAPtool v1.3.0 更新内容

发布日期：2026-04-15

## 版本亮点

- 新增「速决」工具模块
  - 零配置、一键随机的日常决策助手
  - 内置骰子（1-6 随机投掷）
  - 内置硬币（正面 / 反面随机抛掷）
  - 内置 YES / NO（果断给出是/否答案）
  - 支持点击震动反馈，结果直观醒目
- 模块化扩展
  - 新增 `feature:quickdecide` 独立模块，延续项目多模块结构
  - 统一使用 Jetpack Compose + Material 3 风格界面
- 架构与依赖
  - 补充 `material-icons-extended` 依赖，支持更丰富的图标资源

## 其它改进

- 首页工具列表新增「速决」入口
- 导航与工具注册统一接入现有框架
- 整体编译验证通过

## 建议测试项

- 手机竖屏与横屏下「速决」页面的显示与交互
- 骰子、硬币、YES/NO 三个卡片的点击反馈与结果随机性
- 首页点击「速决」后进入与返回流程是否正常
- 平板模式下卡片布局是否正常适配

# BH4HAPtool v1.2.0 更新内容

发布日期：2026-04-15

## 版本亮点

- 新增平板自适应模式
  - 在横屏且长宽比达到阈值时自动启用双栏布局
  - 游戏主画面放在左侧，控制栏和信息栏放在右侧
  - 主画面默认占据约 70% 空间，提升大屏游玩体验

- 抽签功能整合升级
  - 移除独立的“普通抽签”入口
  - 将普通抽签能力合并进“摇一摇抽签”
  - 新增“随机数字抽取”模式，可自定义最小值和最大值
  - 保留手动触发与摇一摇触发两种使用方式
  - 精简页面信息，去掉当前传感器模式栏

- 扫雷体验优化
  - 修复部分大地图和自定义盘面下棋盘超出屏幕的问题
  - 棋盘会根据可用空间自动缩放，减少横向和纵向溢出
  - 更新扫雷首页介绍与页内说明文字

- 推箱子体验优化
  - 修复高地图、宽地图下棋盘容易超出屏幕的问题
  - 棋盘会在可用区域内按比例缩放并居中显示
  - 更新推箱子首页介绍与页内说明文字

## 其它改进

- 补充平板模式开发文档，方便后续子应用统一接入
- 优化多个子应用在大屏设备上的布局结构
- 整体编译验证通过

## 建议测试项

- 手机竖屏、手机横屏、平板横屏下的页面切换和布局表现
- 摇一摇抽签的候选名单模式与随机数字模式
- 扫雷与推箱子在大尺寸地图下的显示是否完整
- 平板模式下左右栏内容滚动和操作是否正常

# BH4HAPtool v1.4.0 更新内容

发布日期：2026-04-18

## 版本亮点

- 工具合集首页升级
  - 首页支持按工具分组展示（随机决策、聚会互动、经典小游戏、效率专注）
  - 支持关键词搜索、收藏筛选、最近使用筛选与平板适配筛选
  - 新增“继续上次”“我的常用”区域，提升高频使用效率
- 合集层能力增强
  - 新增记录中心，统一查看扫雷、俄罗斯方块、推箱子、番茄钟等记录
  - 新增工具使用统计与最近使用记录
  - 新增本地离线成就体系，支持成长反馈
  - 设置页新增首页管理、记录清理与版本更新说明入口
- 活动主持场景扩展
  - 新增幸运转盘：支持权重、抽后移除、结果历史
  - 新增活动记分板：支持 2-8 人/队加减分、撤销、重置、改名
  - 新增轮次管理器：支持随机先手、顺序轮转、淘汰模式
  - 新增活动倒计时：支持多阶段流程与主持模式常亮
  - 新增派对模式入口，提供聚餐破冰/小游戏主持/快速拍板模板

## 稳定性修复

- 修复首页懒加载列表中重复 key 导致的模拟器启动闪退问题
- 提升首页分组与最近使用同时展示时的渲染稳定性

## 建议测试项

- 首页分组、搜索、收藏、最近使用的联动与重启恢复
- 设置页首页管理（收藏/隐藏/排序）后的显示一致性
- 记录中心统计数据汇总正确性与清空能力
- 幸运转盘、记分板、轮次管理器、活动倒计时在手机与平板下的交互表现
- 派对模式模板流程的跳转可用性

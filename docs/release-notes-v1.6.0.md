# BH4HAPtool v1.6.0 更新内容

发布日期：2026-04-19

## 版本亮点

- 新增 `feature:aasplitter` 模块（AA分账器）
  - 支持自动求和与手动总额两种结算模式
  - 支持成员“参与/排除均摊”开关
  - 支持垫付均摊后自动生成转账清单
  - 同时展示应收应付净额与可执行转账路径
- 垫付结算逻辑升级
  - 内部统一按“分”计算，避免浮点误差
  - 支持余数按成员顺序补 1 分，保证总额闭合
  - 手动总额模式提供公共差额提示（未记录支出/退款）

## 记录与成就

- 新增 AA 结算历史（本地最多保留 20 条）
- 新增 AA 统计字段并接入记录中心：
  - 累计结算次数
  - 垫付结算次数
  - 累计结算金额
- 新增 AA 垫付结算成就：
  - 第一笔分账
  - 分账常客（10 次）
  - 结算专家（50 次）

## 建议测试项

- 3-10 人随机垫付场景下，转账结果闭合且可执行
- 排除均摊成员有垫付时应能正常收款
- 小额金额（0.01 / 0.03 / 10.01）下分摊精度正确
- 清空记录后，AA 历史/统计/成就进度被正确重置

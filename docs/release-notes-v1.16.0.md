# BH4HAPtool v1.16 更新内容

发布日期：2026-04-19

## 版本亮点

- AA 分账器新增“垫付金额计算器”
  - 每位成员垫付输入旁新增计算器按钮
  - 支持加减乘除、小数点、清空、退格
  - 计算完成后自动回填到当前成员垫付金额输入框
- AA 垫付均摊结算能力持续完善
  - 支持自动求和与手动总额模式
  - 支持成员参与/排除均摊
  - 结果同时输出应收应付与转账清单
- 新增 GitHub Releases 检查更新能力
  - 设置页新增“检查更新”按钮
  - 支持启动时自动检查更新
  - 发现新版本后可弹窗提示，并跳转浏览器下载

## 记录与成长

- AA 结算历史支持本地留存最近 20 条
- AA 结算统计接入记录中心：
  - 累计结算次数
  - 垫付结算次数
  - 累计结算金额
- AA 垫付成就支持里程碑解锁：
  - 首次结算
  - 累计 10 次
  - 累计 50 次

# BH4HAPtools

一个基于 Android + Jetpack Compose 的轻量工具与小游戏合集，面向日常活动、小型聚会和碎片时间娱乐使用。项目采用多模块结构，当前已支持手机与平板场景，并持续优化大屏横屏体验。

## 项目特点

- 离线可用，打开即玩，适合活动现场快速使用
- 工具与小游戏放在同一入口，支持分组、收藏、最近使用与搜索
- 使用 Jetpack Compose 构建，界面统一，便于后续扩展
- 已加入平板模式，横屏大屏下支持更舒适的双栏布局
- 内置记录中心、成就体系与离线版本更新说明
- 增加派对模式，可按主持模板快速进入活动工具链

## 当前功能

### 1. 摇一摇抽签

- 支持名单抽签和随机数字抽取
- 支持手动触发与摇一摇触发
- 可自定义候选名单与数字范围

### 2. 抓小猫

- 经典围堵玩法，阻止小猫逃到边缘

### 3. 飞盘分组

- 用于随机分队与活动分组
- 支持转盘与抽签两种随机方式

### 4. 扫雷

- 支持经典扫雷玩法
- 支持自定义棋盘大小、雷数与辅助设置
- 已优化大尺寸棋盘的屏幕适配

### 5. 俄罗斯方块

- 经典消行玩法
- 支持速度与震动相关设置

### 6. 推箱子

- 支持关卡游玩、撤销、重开与选关
- 记录最优步数
- 已优化高地图和宽地图的显示适配

### 7. 番茄钟

- 专注计时（默认 25 分钟）与休息计时（默认 5 分钟）
- 支持开始、暂停、跳过、重置
- 自动切换阶段，计时结束震动提醒
- 记录每日完成番茄数
- 计时中保持屏幕常亮

### 8. 速决

- 零配置一键随机决策工具
- 内置骰子、硬币、YES/NO 三种快速决策方式
- 点击即出结果，适合聚会与日常琐事拍板

### 9. 幸运转盘

- 支持自定义候选项与权重
- 支持抽后移除与结果历史
- 适合活动主持、聚会抽选与快速拍板

### 10. 活动记分板

- 支持 2-8 人或队伍记分
- 支持加减分、撤销、重置与快速改名
- 适合桌游、小比赛与聚会主持

### 11. 轮次管理器

- 支持随机先手、顺序轮转和淘汰模式
- 支持导入名单与快速切换当前行动者
- 适合桌游主持与活动排队轮次管理

### 12. 活动倒计时

- 支持多阶段流程配置，如集合、游戏、休息
- 支持开始、暂停、跳过和重置
- 主持场景下可保持屏幕常亮

### 13. 派对模式

- 预置“聚餐破冰”“小游戏主持”“快速拍板”三种流程模板
- 一屏串联分组、随机、记分、轮次和倒计时工具
- 适合现场快速开局

### 14. AA分账器

- 支持自动求和与手动总额两种结算模式
- 支持“排除均摊”成员，同时保留垫付收款能力
- 自动生成应收应付列表与转账清单
- 每位成员垫付金额支持弹窗计算器（加减乘除），结果可自动回填
- 本地保留最近 20 条结算历史

### 15. 工具箱能力增强

- 首页支持按组浏览、搜索、收藏、最近使用和继续上次
- 设置页支持首页管理、记录中心、离线版本更新说明
- 记录中心支持查看扫雷、俄罗斯方块、推箱子、番茄钟等统计与成就

## v1.6.0 更新亮点

- 新增「AA分账器」，支持垫付均摊、排除成员和转账清单
- 结算逻辑统一按“分”计算，保障小额场景下的精度
- 新增 AA 分账历史与统计，并接入记录中心
- 新增 AA 垫付结算成就（首单与累计里程碑）
- 新增垫付金额弹窗计算器，支持四则运算并自动回填

## 运行与构建

### 环境要求

- Android Studio
- JDK 11
- Android SDK 36
- Android 12 及以上设备或模拟器

### 本地编译

```powershell
.\gradlew.bat --no-daemon :app:assembleDebug --console=plain
```
